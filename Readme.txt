Hướng dẫn sử dụng:
Để mở chương trình:
Vào x64 -> Release -> Chạy file Remote Desktop.exe

Lưu ý: Chương trình chỉ chạy được khi 2 máy dùng chung mạng Wifi, Disable tất cả các mạng không liên quan trừ mạng Wifi đang dùng chung với máy còn lại được đảm bảo sự ổn định.


Để kết nối thực hiện như sau:
Server (người được điều khiển): Nhấn nút Open Server.
Client (người điều khiển): Nhập vào IP của Server và nhấn CONNECT NOW (hoặc phím bấm phím Enter)
